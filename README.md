## DeMonStudio
基于JavaFx的开发小工具合集。

目前具备以下功能：

- 字符串替换
- 编码：MD5，DES，AES，Base64
- 二维码生成
- Android App(Apk)签名&渠道标识---基于[SignatureTools](https://github.com/DeMonJavaSpace/SignatureTools)
